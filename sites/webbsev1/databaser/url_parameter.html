Använda url-parametrar

Vi lägger till url parametrar till en länk på följande sätt: 

<a href="somepage.php?id=2">en länk till en sida</a>

Vi kan lägga till flera url parametrar genom att separera dem med ett &-tecken:

<a href="somepage.php?post=7&id=2">en länk till en sida</a>

URL parametrarna hamnar i superglobala arrayen $_GET, och vi kan hämta ut värden:

$id = $_GET[id];

Värdet som hämtas ut kommer alltid att vara en sträng.

Denna teknik används för att sidan ska kunna visa rätt innehåll beroende på användarens val. Exempelvis:
Låt säga att vi vill visa en lista på alla registrerade användare i bloggen, vi vill dessutom att man ska kunna klicka på en användare för att besöka just den användarens blogg. Sidan kallar vi för view.php.
Vi hämtar ut användarna från databastabellen user, vi stegar igenom resultatet och för varje användare så genererar vi en länk där vi skickar med värdet för id. Bloggens titel används till länkens text i detta exempel, vi kapar dessutom titeln, så att den inte blir mer än 30 tecken lång.

//Uppkoppling mot databas....

$query = 'SELECT id, title FROM user';
$result = mysqli_query($connection, $query);
foreach ($result as $value)
{
    $id = $value['id']; // Hämtar bloggarens id
    $title = substr($value['title'], 0, 30); // Hämtar titel på bloggen, max 30 tecken ska visas
    echo "<a href='view.php?id=$id'>$title</a><br>"; // Skriver ut länken
}

//Koppla ner mot databas....

Samma teknik används för att till exempel visa en lista med inlägg, när man klickar på ett inlägg så skickar man med bloggarens id och inläggets id, på så viss kan man hämta och visa rätt inlägg. Om inget inlägg är valt så visas det senast skrivna inlägget.
