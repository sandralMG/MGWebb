DOM Events

Hur man skapar interaktivitet

Events kan översättas till händelser på svenska och det här avsnittet handlar om hur man kan göra så att en funktion kan triggas när en händelse inträffar. Exempel på händelser kan vara:

Känner av när webbsidan är färdigladdad
Skicka iväg formulärdata (submit)
Musklick på en knapp
Fokus på ett textfält i ett formulär
Reagera när musen förs över något
Tangenttryckning

M h a DOM kan vi koppla specfika händelser till ett visst element på sidan, exempelvis en knapp, men det kan lika gärna vara en bild eller annat element. När händelsen för ett element triggas körs en specifik funktion som interaktivt ändrar någonting.

Det finns många olika händelser. Här finns alla händelser, ca 300 st, som är möjliga genom DOM enligt Mozilla Developer Network (MDN). Du behöver självklart inte veta om alla dessa utan endast ett fåtal, men det är bra att veta om att de finns :-). Vi börjar med kanske den vanligast händelsen - musklick.

Hur?

Tre delar måste kopplas ihop: Elementet på webbsidan, händelsen och funktionen ska ska utföras när händelsen inträffar.

För att lägga till en funktion som kan "lyssna" när en viss händelse inträffar använder vi metoden addEventListener. Metoden kopplas till det element som ska triggas. I argumentet till funtionen anges vilken funktion som ska anropas om händelsen inträffar - i detta fall "click", d.v.s ett musklick.

Alternativ 1:

Detta är ett skrivsett på att använda en namngiven funktion som argument till addEventListener. Om vi exempelvis skulle behöva använda vår funktion på ett flera ställen i koden är detta skrivsätt att föredra.

Men man kan också skriva på formen där en anonym funktion anges som argument i funktionen. Detta är ett behändigt sätt, då koden som utförst för händelsen är samlad på ett ställe.

Alternativ 2:


Vilken av dessa två alternativ du använder är upp till dig. Det är bra att du känner till skillnaden.

En funktion som anges som argument till en annan funktion och som exekveras först när den yttre funktionen åberopas kallas för övrigt för callback-funktion.
