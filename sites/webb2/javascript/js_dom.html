Javascript Dokumentobjektsmodellen (DOM)

Dokumentobjektsmodellen brukar förkortas DOM och är det som gör det möjligt att göra webbsidor interaktiva. De flesta webbsidor utför instruktioner som körs av ett Javascript. Exempelvis när man ska klicka fram nya bilder i ett bildspel, en interaktiv navigationsmeny eller att få återkoppling om ett formulär är rätt ifyllt eller inte. Denna funktionalitet vilar på att Javascript kan få åtkomst till webbsidan genom DOM för att interaktivt kunna få det förändrade innehållet eller få information om vad använderen har matat in på sidan.

DOM är alltså själva gränsnittet som tillåter Javascript (och även andra programmeringsspråk) att ändra eller manipulera innehåll, struktur eller styling till ett html-dokument. I detta avsnitt ska vi lära oss grunderna hur vi kan hitta element i en html som vi sedan vill kunna ändra exempelvis textinnehåll, attribut till element eller styling.

Vad är DOM?

En webbläsare är ett program som tolkar, eller bättre uttryck intepreterar medföljande HTML och CSS. Den renderar webbsidan så att den visar det den ska. Förutom att webbläsaren renderar webbsidan så skapas en representation av dokumentet i webbläsarens - denna representation är Dokumentobjektsmodellen. Denna represenation är en modell som tillåter Javascript att komma åt alla element och dess innehåll av webbsidans dokument där varje del i modellen är ett objekt.

Dokumentobjektsmodellen är ett långt ord, men dess innebörd handlar om att webbsidan lagras som en represantation eller modell i det s.k document-objektet vilket innehåller hela representation eller modellen för webbsidan. Om vi skriver document i consolen får vi ett svar på webbsidans struktur. Den ser ju likadan som själva html-koden - men det är detta som är själva DOM som vi kan interagera och förändea med Javascript.

Om vi istället skriver document.body får vi åtkomst endast till body-delen av webbsidan.

DOM brukar i många fall förklaras med en trädstruktur som innehåller alla objekt som finns på webbsidan. Varje del i trädet, eller objekt, brukar kallas nod. Som man ser på bilden är alltså document roten i trädet som håller alla noder. Detta gör det möjligt att hitta det element man vill manipulera.

M.h.a DOM kan vi hitta det elementet som vi är intresserade av. Exempelvis rubriken h1 och ändra denna till en annan med Javascript. Hur vi gör det är alltså grunden att skapa interaktivitet med Javascript, vilket vi nu ska lära oss!

DOM Selektorer - metoder för att hitta element

Det finns olika metoder att hitta element med Javascript. Vi kommer framförallt att lära oss dess tre. Det finns även fler, men man kan klarar sig bra att först lära sig dessa.

  • getElementById()
  • querySelector()
  • querySelectorAll()

Vi börjar med getElementById(). Denna metod tar en sträng som argument och returner det matchande elementet med det angivna id:et.

Här är ett exempel hur man kan hitta ett element med ett visst id. Referensen till objektet eller noden i DOM lagras i variabeln element. Notera här att det står document.getElementById(). Varför? Jo vi vill ju leta i document-objektet som innehåller hela webbsidan.

Om man vill ändra nodens innehåll med en ny text kan man använda medtoden textContent.


Sedan finns även querySelector() som är väldigt praktisk eftersom det argument den tar in är samma som CSS Selektorer. Denna returnerar det första elementet med matchande selektor den stöter på.


querySelectorAll() returnerar en list på alla noder som matchar CSS-selektorn

Ändra i DOM - maniplera de element vi hittar

För de element vi hittar kan vi:

  • Ändra innehåll (text/element)
  • Ändra style
  • Ändra attribut (src, href, etc)
  • Lägga till eller ta bort classer
  • m.m om man vill lära sig mer...

I exemplen ovan visas hur textContent används för att ändra textinnehållet i en nod. En annan metod som ändrar innehållet i ett element är innerHtML där skillnaden ligger i att man även kan inkludera html-taggar.

Vi kan även ändra utseendet genom CSS och style för element

Det är dock inte rekommenderat att förändra style om man ska utföra omfattande förändringar i style.

Istället för att ändra style med Javascript kan vi använda oss av lägga till, ta bort eller toggla ett class attribut på det element vi vill förändra style. Detta göra med classList.add, classList.remove eller classList.toggle

Vi kan också hämta eler ändra ett elements attribut med getAttribute() och setAttribute(). Bra exempel på detta är att komma åt "href" i a-taggen eller ändra en bildkälla med "src" i img-taggen.
