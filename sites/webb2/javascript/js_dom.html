Javascript Dokumentobjektsmodellen (DOM)

Dokumentobjektsmodellen brukar förkortas DOM och är det som gör det möjligt att göra webbsidor interaktiva. De flesta webbsidor utför instruktioner som körs av ett Javascript. Exempelvis när man ska klicka fram nya bilder i ett bildspel, en interaktiv navigationsmeny eller att få återkoppling om ett formulär är rätt ifyllt eller inte. Denna funktionalitet vilar på att Javascript kan få åtkomst till webbsidan genom DOM för att interaktivt kunna få det förändrade innehållet eller få information om vad använderen har matat in på sidan.

DOM är själva gränsnittet som tillåter Javascript (och även andra programmeringsspråk) att ändra eööer manipulera innehåll, struktur eller styling till ett html-dokument. Vi använder ju oss av Javascript och det är genom DOM som vi kan koppla instruktioner i vår kod till en webbsida. I detta avsnitt ska vi lära oss grunderna hur vi kan hitta element i en html som vi sedan vill kunna ändra exempelvis textinnehåll, attribut till element eller styling.

Vad är DOM?

En webbsida innehåll ett html-dokument. Webbläsare som visar webbsidan är ett program som tolkar, eller bättre uttryck intepreterar den HTML och CSS som gör att webbsidan visas som den ska. Med andra ord säger man att webbläsaren renderar innehållet, strukturer och utseendet.

Förutom att webbsidan renderas av webbläsaren så skapas en representation av dokumentet i webbläsarens - denna representation är Dokumentobjektsmodellen. Denna represenation är en modell som tillåter Javascript att komma åt alla element och dess innehåll av webbsidans dokument där varje del i modellen är ett objekt.

Nu ska vi titta på webbsidans represention eller modell som den tolkas i webbläsaren. Dokumentobjektsmodell är ett långt ord, men dess innebörd handlar om att webbsidan lagras som en represantation eller modell i det s.k document-objektet vilket innehåller hela representation elelr modellen för webbsidan. Om vi skriver "dokument" i consolen får vi ett svar på webbsidans struktur. Den ser ju likadan som själva html-koden - men det är detta som är själva DOM som vi kan interagera och förändea med Javascript.

Om vi istället skriver document.body får vi åtkomst endast till body-delen av webbsidan.
