Att använda sig av DOM med Javascript

Syftet med att använda sig av DOM är att man kan förändra en webbsidans innehåll eller utseende enbart genom Javascript. Detta är primärt användbart när vi kommer att lära oss hur man kan trigga händelser med att klicka på en knapp eller interaktioner från en användare. Vi har inte kommit till att lära oss detta ännu utan i detta avsnitt kommer vi enbart fokusera på man kan ändra eller manipulera element genom DOM.

Själva proceduren är att man alltid först vill HITTA en referens till det element som man sedan vill ÄNDRA . Exempelvis att ändra textinnehåll, innehåll med html eller ändra i style genom CSS. Man kan också ändra attribut till element.

Att hitta en referens till ett element gör genom att välja en lämplig metod, s.k. DOM Selektorer.

HITTA element

Det finns olika metoder att hitta element med Javascript, vilket kallas för DOM Selektorer. Vi kommer framförallt att lära oss dessa tre. Det finns även fler, men man kommer långt med endast dessa.

  1. getElementById()
  2. querySelector()
  3. querySelectorAll()

Här nedan förklaras skillnaden mellan dessa.

1. getElementById()

Denna metod tar en sträng som argument och returner det matchande elementet med det id som är angivet i elementet.

Här är ett exempel hur man kan hitta ett element med ett visst id. Referensen till objektet eller noden i DOM lagras i variabeln element. Notera här att det står document.getElementById(). Varför? Jo vi vill ju leta i document-objektet som innehåller hela webbsidan.

Om man vill ändra nodens innehåll med en ny text kan man använda metoden textContent.


2. querySelector()

Denna metod är praktisk, till skillnad från getElementById(), då det argument man tar in är samma som CSS Selektorer. Denna returnerar det första elementet med matchande selektor den stöter på.


3. querySelectorAll()

Denna metod är ungefär samma som querySelector() men skillnaden ligger i att den returnerar en lista på alla noder som matchar CSS-selektorn. I exemplet nedan när man vill hämta alla "li" finns det flera referenser än en. I dessa fall är denna metod att föredra.

ÄNDRA element

För de element vi hittar kan vi:

  • Ändra innehåll (text/element)
  • Ändra style
  • Ändra attribut (src, href, etc)
  • Lägga till eller ta bort classer
  • m.m om man vill lära sig mer...

I exemplen ovan visas hur textContent används för att ändra textinnehållet i en nod. En annan metod som ändrar innehållet i ett element är innerHtML där skillnaden ligger i att man även kan inkludera html-taggar.

Vi kan även ändra utseendet genom CSS och style för element

Det är dock inte rekommenderat att förändra style om man ska utföra omfattande förändringar i style.

Istället för att ändra style med Javascript kan vi använda oss av lägga till, ta bort eller toggla ett attribut som exempelvis class=".some-class" på det element vi vill förändra style. Detta kan göras med classList.add(), classList.remove() eller classList.toggle(). Då kommer elementet bli tilldelat stilregler som finns angivna i CSS som hör till den selektorn.

Vi kan också hämta eler ändra ett elements attribut med getAttribute() och setAttribute(). Bra exempel på detta är att komma åt "href" i a-taggen eller ändra en bildkälla med "src" i img-taggen.
