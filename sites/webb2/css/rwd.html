Responsive Web Design

1. Sätta viewport

För att sida ska fungera responsivt måste denna rad finnas med i <head>. Den är viktigt för att en webbläsare ska kontrollera sidans dimension och skalning. 

<meta name="viewport" content="width=device-width, initial-scale=1">

2. Fluid layout

I en fluid layout varierar storleken på en layout i förhållande till skärmstorleken. Det nås genom att använda relativa enheter, som % m.fl istället för pixlar. Det innebär både att layout, sin grid, samt att bilder och text kan variera i förhållande till skärmstorleken och anpassas till den enhet man använder (desktop, mobil, etc. ).
I de nya teknikerna som CSS Grid och Flexbox följer många anpassningar till responsiv web design. I CSS Grid finns exempelvis enheten fr och minmax(). I Flexbox är mycket "flexbilt" och boxar fyller det specificerade utrymmet i layouten.

Ändra storlek på text responsivt

Best practice för att ändra storlek på olika text element är att använda sig av enheterna %, em, rem. Det finns lite olika angreppsätt om vilken som är bäst. I detta exempel visar jag hur man kan använda rem som baseras på font-size relativt till rot-elementet, d.v.s html. Genom att sätta all text samt margin/rem baserade på denna kan man förändra alla element genom att endast förändra font-size i roten.

Desktop vs Mobile-first

Inspiration från första förespråkaren till mobile-first 

<!DOCTYPE html>
<html lang="en">
<head>
<meta charset="UTF-8">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<link rel="stylesheet" href="style.css">

<title>Desktop first vs. Mobile first</title>
</head>
    <body>
    <h1> INNEHÅLL</h1>
    <p> Mer innehåll...</p>
    </body>
</html>

Att använda Desktop-first

I detta fall används max-width i mediaquery.

/*DESKTOP LAYOUT*/
/*  Fråm 600px och uppåt */

body {
    background-color: #ccf;
}

h1 {
    font-size: 22px;
    display: flex;
    justify-content: center;
    align-items: center;
}


/*FÖRÄNDRAD LAYOUT FÖR MOBIL*/
/*Från 0px till 600px*/
@media screen and (max-width: 600px) {
    body {
    background-color: #cfc;
    }
    h1 {
    font-size: 16px;
    }
}

Att använda Mobile-first

I detta fall används min-width i mediaquery.

/*LAYOUT FÖR MOBIL */
/*Från 0px till 600px*/
body {
background-color: #cfc;
}

h1 {
font-size: 16px;

}



/*FÖRÄNDRAD LAYOUT FÖR DESKTOP*/
/* Från 600px och uppåt */
@media screen and (min-width: 600px) {
body {
    background-color: #ccf;
}

h1 {
    font-size: 22px;
}
}
