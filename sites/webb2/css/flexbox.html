CSS Flexbox

Flexbox är ett modernare sätt att skapa layout på webbsidor. I Webbutveckling 1 använde vi oss av floats för att skapa layout. När vi exempelvis ville ha en vänsterkolumn eller att en bild skulle ligga till höger. Floats har funnits med sedan början av HTML/CSS, men för vissa situationer innebär det oänskade resultat som kräver omständig lösnign. Exempelvis att behöva använda clear på floats för att nästa element ska följa nedanför och "the clearfix-hack" eller att få kolumner bredvid varandra på samma höjd, etc (läs mer här om du är nyfiken). Flexbox är inte till för att helt ersätta floats, men det ger oss bättre lösnignar på de "problem" som floats skapar och ger oss nya möjligheter till att skapa en flexibel layout på ett smidigare sätt.

Sidan "Can I Use" visar hur Flexbox supportas i olika webbläsare, vilket visar att felxbox har bra support i nuvarande webbbläsare. Det är endast för tidigare versioner av Internet Explorer som man bör implementera en fallback för.

Att lära sig Flexbox

Flexbox är inget eget språk utan det handlar om nya egenskaper i CSS som vi kan applicera på våra element. Det är enkelt att komma igång med Flexbox, man använder egenskapen   display:flex;  på det element man vill applicera reglerna för Flexbox på, exempelvis en meny, ett bildgalleri o.s.v. Det svåra är inte att komma igång med flexbox utan att lära sig alla olika inställningar ett flexbox-element kan ha och vad de gör i sammanhanget.
