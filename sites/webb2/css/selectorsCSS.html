CSS Selektorer

Nödvändiga att kunna i kursen

1. *

Universalselektorn

Denna selektor väljer alla element på sidan - all textfärg blir grön. 

* {
    color: green;
}

2. X

Elementselektorn

I detta fall blir alla länkar lila och alla paragrafer teckenstorlek 20px. 

a { color: purple; }
p { font-size: 20px; }

3. X, Y

Gruppselektorn

Separeras med ett kommatecken. Alla div's samt alla paragrafer får typsnittet Arial och storlek 22px. Fler än två element kan väljas. 

 div, p {
        font-family: Arial;
        font-size: 22px;
    }

4. #X

Selektor för id

Elementet med id="container" får färgen aqua som bakgrundsfärg. Id kan endast tillhör ett element. 

 #container {
            background-color: aqua;
        }

5 .X

Selektor för class

Alla element med class="gallery" får bredd 50% utifrån upptaget område av elementet. Class kan till skillnad från id sättas på flera element än ett. element. 

 .gallery {
                width: 50%;
            }

6. X Y

Arvselektor

Du använder denna när du vill komma åt ett element som är underelement till ett annat, d.v.s barn (child) till förälderelementet (parent).

I detta fall får alla paragrafer med class="info" violet textfärg.

 .info p {
            color: violet;
        }

Här får alla länkar i varje list-element i en oordnad lista en grå bakgrund.

 ul li a {
            background-color: grey;
        }

Pseudo-klasser

För psedo-klasser sätter man kolontecknet ( : ) efter selektorn. De används för applicera en regel för ett valt tillstånd, exempelvis när en användare drar musen över ett visst element.

8. X:hover

När man gör mouse over element aktiveras regeln. I detta fall blir bakgrundfärgen lila när man "hovrar" över en länk. Bra att använd till knappar och menyer.

a:hover {
                background-color: yellow;
            }

7. X:visited

I detta fall blir alla av användaren besökta länkar gråa. 

a:visited {
     color: grey;
}

9. X:focus

När ett element klickas eller vid "tab"-tangenten når elementet. Användbart vid att styla textrutor (input) i formulär.

input:focus{
     border: 2px solid green;
}

10. X:link

Alla länkar som ännu inte blir besökt är röda 

a:link{
     color: red;
}

Bra att kunna i kursen

12. X[type="value"]

Selektorn för att sätta en regel till ett visst attribut med ett visst värde. I detta fall får endast alla textrutor en grå bakgrund (inte andra som ex checkbox, m. fl.)

input[type="text"] {
     background-color: lightgrey;
}

21. X:nth-of-type(n)

Väljer det n:te elementet som finns inuti förälderelementet. I detta fall blir textfärgen grön för det andra p-elementet inuti det överordnade elementet.

p:nth-of-type(2) {
     color: green;
}

13. X > Y

Liknar arvselektor X . Skillnaden är att den endast väljder de element som är första barnet till föräldern. Här väljs alltså bara de p som har förälder div, inte de som är underornade p. 

div > p {
     background-color: blue;
}

16. X ~ Y

Du kanske använder någon gång i kursen

Mer info kommer. Se annars länk nedan.

18. X:first-child

Mer info kommer. Se annars länk nedan.

19. X:nth-child(n)

Mer info kommer. Se annars länk nedan.

20. X:nth-last-child

Mer info kommer. Se annars länk nedan.
