Introduktion till Webbutveckling

Webbutveckling är en samlingsterm för allt runtomkring uppbyggnaden av en webbplats. Det handlar både om att skapa webbplatser eller webbapplikationer och inkluderar såväl ta fram en design för sidan utseende (exempelvis i Photoshop eller på papper), ta fram protoyper för att pröva webbsidans användbarhet, till att koda själva sidan för att den ska fungera i olika webbläsare. Det kräver även att man testar webbsidan så att den fungerar som den ska i olika webbläsare, men även att den är optimerad så att den kan laddas hem snabbt. Det är även viktigt att anpassa webbplatsen till alla besökare, även de som använder hjälpmedel som skärmläsare för blinda. Man bör även ha kännedom om de tekniska aspekterna på webben, exempelvis om hur datorer kommunicerar med varandra och olika säkerhetsaspekter som man bör ta hänsyn till. Till sist hur man publicerar sin webbsida på world wide web och hur man underhåller :-)

Webbutveckling kan alltså vara allt från att skriva enkel HTML i en textfil till att bygga komplexa webbapplikationer. Moderna webbsidor bygger på tre grundläggande tekniker som vi mer eller mindre kommer gå igenom i kursen.

HTML

Är grunden för att skapa webbsidor. HTML står för Hyper Text Markup Language och är ett enkelt språk som definerar hur webbsidan ska struktureras. Exempelvis, “det här är en rubrik” eller “lägg in en bild här”. Det är html-filen som laddas in och tolkas av webbläsaren för att kunna visas som den ska för användaren. Man kan säga att HTML är “substantiven” för webbsidan.

CSS

Står för Cascading Syle Sheets och ger sidan utseende. Exempelvis gör den här “rubriken blå” eller “lägg den här bilden tlll höger på sidan”. Man kan även göra enklare animationer och effekter med CSS, som exempelvis byta bakgrundsfärg när man drar musen över. Om HTML är substantiven så kan CSS beskrivas som “adjektiven”.

Javascript

Är ett programmeringspråk till skillnad mot HTML/CSS som är märkspråk. Javascript ger sidan interaktivitet och kan tillföra logik. Exempelvis en lista i menyn ska synas när jag klickar tll att göra mer avancerade applikationer till att göra spel. Eftersom Javascript ger funktionalitet, gör så att det händer saker, kan det ses som “verben” för sidan. I Webbutveckling 1 ingår endast en orientering om Javascript. Det området lärs ut mer grundligt i Webbutveckling 2.
