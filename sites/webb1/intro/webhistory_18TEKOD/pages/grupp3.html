CSS, Javascript, Flash och HTML

CSS

CSS (Cascading Style Sheets) är ett enkelt språk som används för att beskriva presentationen av sidan. Hur sidan ser ut, färg, typsnitt, textstorlek osv. är alltså vad CSS förändrar.

Historia

Innan CSS bestod alla hemsidor av svart text på en vit bakgrund, när W3C (World WIde Web Consortium) släppte CSS1 december 1996 så kunde man ändra på väldigt få saker men det förändrade webben. Eftersom man kunde ändra utseendet på en hemsida utan att ändra i HTML-dokumentet. I november 1997 släpptes nästa version av CSS, CSS2 kom med uppgraderingar och var mer dynamiskt än dess företrädare. CSS2.1 kom ut 2004 och fixade många fel från den tidigare versionen, och introducerade nya funktioner, man kunde nu ha parent-child relationer. Den version av CSS vi alla är bekanta med nu är CSS3, den största skillnaden är moduler. Moduler har sina egna funktioner och förmågor vilket expanderar CSS ännu mer än förut. CSS4 utvecklas inte eftersom det största fokuset är att skapa nya moduler istället.

Javascript

Javascript är ett prototyp-baserat skriptspråk som är dynamiskt, svagt typat. Detta betyder att man inte behöver ange datatyper när man skapar variabler. Istället försöker javascript “fatta” vad variabeln är till för och använda den på det mest troliga viset. javascript är ett programmeringsspråk som till stor del används för att höja interaktiviteten mellan en webbplats och dess användare. Javascript koden körs direkt webbläsaren till skillnad från andra programmeringsspråk.

Historia

Företaget Netscape grundades 1994 och skapade en av de första webbläsarna. De rekryterade Eich år 1995 eftersom de ville att han skulle skapa ett programmeringsspråk för den webbläsaren. Netscapes mål var att Javascript skulle fungera som ett “glue language” för deras webbläsare. Javascript skapades väldigt snabbt och har fått utstå mycket kritik för att det inte var väldesignat från grunden. Trots detta blev det snabbt accepterat som standardspråket för att köra kod i webbsidor.

Flash

Adobe Flash eller enbart kallat Flash, är ett Datorprogram som används till att skapa animationer, spel och så kallade Rich Internet Applications (RIA) för webben. För att visa Flash-grafik i en webbläsare krävs ett insticksprogram som Adobe Flash Player eller Gnash. Insticksprogrammet är det som krävs för att en annan enhet ska läsa Flash. Medans Adobe är det man arbetar i för att utveckla applikationer mm i.

Historia

Föregångaren till Flash hette SmartSketch och skapades av Jonathan Gay. År 1995 nylanserades programmet vid namnet ”FutureSplash Animator” och blev populärt av företag som ville skapa animationer webben. Bland kunderna fanns då företag som Disney, MSN och Fox. November 1996 blev FutureSplash köpt av Macromedia, FutureSplash Animator lanserades som Macromedia Flash 1.0. Under åren tillkom en mängd nya funktioner och användningsområden. Populariteten ökade och 2005 var Flash Player det vanligaste insticksprogrammet för multimedia. År 2005 köptes Macromedia upp av Adobe och Flash blev en del av deras professionella designserie. Samma år lanserades Youtube och från början var sidans videoformat Flash video vilket ökade Flashs popularitet ytterligare. År 2007 lanserade Apple iPhone och meddelade att den inte hade stöd för Flash. Steve Jobs menade att Flash hade dålig säkerhet, drog för mycket batteri och var dåligt kodat. Detta medförde en kraftig nedgång i Flashs popularitet. Flash används idag främst för att utveckla mobilspel med Adobe AIR.

html

HTML står för “Hypertext Markup Language”. Hypertext betyder att det finns text ihop med hyperlänkar. Hyperlänkar är för att hoppa från ett till ett annat ställe utan att söka sig fram, utan man tar sig dit med en klick. WWW=World Wide Web är det som gör så att alla datorer förstår/läser av varandra genom samma filer som ligger i serverar.

Historia

Sir Tim Berners-Lee är en brittisk data-vetenskapare. Han är född i London, och hans föräldrar var unga data-vetenskapare som arbetade på en av de första datorerna. Sir Tim märkte att de hade svårt att dela information med varandra i världen, han ville göra något åt det.
