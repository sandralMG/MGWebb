90-talets CSS och Javascript

CSS

Cascading Style Sheet (CSS) som på svenska kallas för stilmall, är ett språk som beskriver presentationsstilen för ett strukturerat dokument som exempelvis; typsnitt, färger och textstorlekar etc. Namnet kommer från en författares möjlighet att definiera utseendet av ett flertal olika dokument med olika innehåll.

Historia CSS

CSS plockades fram av norrmannen Håkon Wium Lie år 1994 då han arbetade på CERN tillsammans med Tim Berners-Lee (grundaren av WWW) och Robert Cailliau. World Wide Web Consortium (W3C) hade tidigare fått många förslag på liknande program men standardiseringsorganet valde att använda Wium Lies. 1996 släpptes den första officiella versionen av CSS och den hette CSS1 (Level 1). Efter komplikationer då CSS1 inte svarade så började CSS2 utvecklas och blev en verklighet i början av November 1997. 2004 lanserades en uppdatering på CSS2 som kallades CSS 2.1 och fixade en del buggar och felaktigheter. CSS3 rekommenderas 1998 och utvecklas än idag. Produktionen av CSS4 avbröts då utvecklarna istället fokuserar på att skapa moduler i level 1-5 inom CSS3 istället för att skapa en ny version av programmet inför varje ny level. CSS3 är den senaste (och sista) uppgraderingen av CSS.

HTML

HTML (HyperText Markup Language) 1.0 utvecklades sent 1990, och man kunde inte göra mycket mer än att få upp simpla textrader på webben med det. Sent på år 1993 och tidigt 1994 kom drafterna för HTML 2.0, med en version av Tim Berners-Lee och Dan Connolly, och HTML+ av Dave Raggett och ytterligare en grupp, kallad IETF, som konkurrens. HTML 2.0 blev aldrig en godkänd standard.

Javascript

Javascript utvecklades ursprungligen av Brendan Eich, då kallades det för Mocha men namnet ändrades senare till Livescript. Men i och med att Netscape samtidigt började stödja Sun Microsystems programspråk Java med syftet att kunna ha miniprogram inuti webbläsaren, valde man att kalla det för Javascript främst i marknadsföringssyfte. Förutom att Java och Javascript har liknande namn så har de ingenting med varandra att göra. De delar bara samma program språk, C:s Syntax. Javascript dök för första gången upp i Netscape Navigator 2.0B3 som släpptes år 1995. Microsoft skapade en egen version av Javascript, som döptes till JScript för att undvika marknads-intrång. Den grundläggande idén bakom Javascript var att skaparen av en webbsida skulle ha ett verktyg för att skapa mer interaktivitet på sin sida. Javascript är ett prototyp baserat skriptspråk. När javascript används i webbläsaren arbetar det mot ett gränssnitt som kallas Document Object Model (DOM). Exempel på användningsområden är kontroll av ifyllda fält innan formulär skickas till en server, funktioner för att visa eller dölja delar av en sida och en växling mellan annonsbilder. Det kan även användas för mer avancerade funktioner som för spel och bildbehandling.
