Introduktion till CSS

CSS är ett språk som sätter regler för hur HTML-element ska visas eller ha ett visst utseende, som exempelvis färg, typsnitt, storlek, layout m.m. Om HTML står för innehållet på webbsidan/webbplatsen, tillför CSS sidans utseende - den styr sidan design. CSS är en förkortning för Cascading Style Sheets och översatt till svenska kallas det för en sidan stilmall.

Hur en stilmall fungerar:

  1. Börja med ett html-dokument som är uppmärkt
  2. Skapa ett nytt dokument med filändelsen .css (d.v.s en stilmall)
  3. Skriv CSS-regler för ett eller flera element i din stilmall
  4. Koppla stilmallen till ditt html-dokument

Oftast arbetar man med endast en stillmall til sin webbplats. Ha som vana att döpa denn stilmall till style.css

Skriva en CSS-regel

En stilmall innehåller en eller flera instruktioner om hur html-element på sidan ska visas. Det första man bekantar sig med när man lär sig är hur man skriver instruktioner i en stillmall är följande:

Följande regel säger att alla paragrafer p ska ha en blå färg på texten. Med andra ord är det selektorn som identifierar vilka html-element som kommer påverkas av regeln. Deklarationen innehåller vad som ska ändras med egenskap och tillhörande värde.

En deklaration kan även ha ett block av olika regler. 

p {
    color: blue;
    font-family: Arial;
    font-size: 18px;                                
}

Här får även alla p-element på html-sodan typsnittet Arial samt teckenstorleken 18px.
På engelska motsvarar selektor - selector, deklaration - declaration, egenskap - property, värde - value. Det är bra att kunna när ni läser i engelska guider om CSS.

Nu har vi använt oss av den s.k element-selektorn. Men det finns flera sätt vi kan använda selektorer som är viktiga att kunna inom webbutveckling. Läs mer här.

Exempel på HTML med stillmall (CSS)

Vi utgår från en uppmärkt HTML enligt exemplet nedan.
Vi länkar till en css-fil i <head>-elementet med <link>-elementet.Här använder vi oss av extern stilmall (se andra sätt här ). 

<!-- En extern länk till en stilmall style.css -->
<link rel="stylesheet" type="text/css" href="style.css">

    <!DOCTYPE html>
    <html lang="sv">
        <head>
        <meta charset="utf-8">
        <link rel="stylesheet" type="text/css" href="style.css">
        <title>Min första stylade sida</title>
        </head>
    <body>

        <h1>En huvudrubrik</h1>                     
        <p>Det här är innehållet - en text jag vill
        ha med på min sida </p>
        
        <div id="info">
            <h2> En underrubrik</h2>  
            <p> Lite mer text</p>
        </div> 

        <div class="gallery">                        
            <img src="minbild.jpg" alt="Min bild">  
        </div> 
        
    </body>
    </html>

Exempel på CSS-regler i stillmallen

I filen style.css har jag angett följande CSS-regler. Du kan visa HTML-sidan som stylad genom att trycka "Visa sida" i HTMl-exmplet ovan.


    body {
        background-color: lightblue;
    }
    
    h1, h2 {
        font-family: Verdana, Geneva, Tahoma, sans-serif;
        color: green;
    }
    
    p {
        font-family: Arial, Helvetica, sans-serif;
        font-size: 22px;
        color: purple;
    }
      
    #info {
        background-color: beige;
        border: 2px solid #000000;
        width: 30%;
    }
      
    .gallery {
        margin-top: 15px;
        border: 2px solid yellow;
    }
      
    .gallery img {
        padding: 10px;
    }
  • Hela sidan, d.v.s <body>-elementet har fått en ljusblå bakgrundsfärg
  • Både rubrik h1 och h2 har fått ett nytt typsnitt som är Verdana, så länge den finns installerad på användarens dator, annars Geneva o.s.v. Färgen på texten är satt till grön. För att välja fler element, d.v.s en gruppselektor, separerar man dem med ett kommatecken.
  • Paragraferna p ges ett typsnytt Arial (i första hand) samt ökad texkenstorlek till 22px. Texten får en färg lila.
  • Här anropar vi vår <div>-element med id="info". En id har selektor # framför namnet. Här ges alltså hela innehållet innanför <div>-elementet en bakgrund beige samt en svart kantlinje på 2px bred. Området blir också 30% av hela skärmbredden
  • Här anropar vi vår <div>-element med class="gallery". En class har selektor . (punkt) framför namnet. Det området får en gul kant på 2px. Vi gör även lite mellanrum till elementet ovan på 15px med egenskapen margin-top
  • Till sist anropar vi bilden inuti <div>-elementet med class="gallery" och ger lite mellanrum inuti boxen med egenskapen padding på 10px

I exemplet har vi alltså tagit upp några nya selektorer som gruppselektor och för id och class. Läs mer på CSS Selektorer
Egenskaperna margin och padding kommer vi även förjupa oss mer i i avsnittet Boxmodellen.
