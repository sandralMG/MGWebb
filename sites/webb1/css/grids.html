CSS Grids

CSS Grid Layout är en relativt ny webbteknik som gör det möjligt att arbeta med gridbaserad layout i ren CSS. Att fördela innehåll i rader och kolumner är i och för sig inget nytt sätt att arbeta på. Tidigare har man behövt använda stöd av ramverk såsom Boostrap eller egna definierade rader och kolumner. Att kunna skapa en flexibel och responsiv gridlayout i CSS har länge varit önskvärt och är möjlig i skrivande sekund om man ser till CSS Grid support för de största webbläsarna som Chrome, Firefox, Safari och Edge.

CSS Grid hanterar både X-axel och Y-axel samtidigt - med andra ord en tvådimensionnell layoutteknik. Det innebär att man kan arbeta med rader och kolumner på ett mycket smidigare sätt än tidigare. Det ska bli spännande att se utvecklingen av CSS Grid och vad den möjliggör inom layout och design. Kolla gärna in Experimental layout lab by Jen Simmons och 10 Creative Examples of CSS Grid in action .


Grid terminologi

Hur CSS Grid fungerar

  1. Använda display:grid för att göra ett element till en grid container.
    På så sätt blir även barnen till föräldraelementet till grid items
  2. Specifera rader och kolumner för sin grid
    Antingen sätts de explixit med ett fixt antal eller implicit om en grid ska vara flexibel att hantera nya rader/kolumner.
  3. Tilldela varje grid item en specifik yta på vår grid.
    Om de inte tilldes en viss yta kommer de att flöda i den normala ordningen.

Det finns många olika sätt i CSS Grids att skapa rader/kolumner och definera grid areas. Vi börjar med att lära oss de egenskaper för en grid cointainer och grid items som är absolut mest nödvändiga.

CSS Grid vs Flexbox

Eftersom du har lärt dig grunderna i Flexbox kanske du undrar om CSS Grid helt ersätter den andra layouttekniken?
I huvudsak kan man säga att Flexbox är bra på endimensionell layout och CSS Grid för tvådimensionell layout. Det finns också en del egenskaper som både Flexbox och CSS Grid kan göra. så vad ska man välja? Läs gärna här eller här om det området. Generellt är vägledningen att använda CSS Grid för övergripande layout och Flexbox när man ska ordna innehåll inom ett element. Exempelvis: Ordna elementen inom en header med flexbox, men göra hela layouten med CSS Grid.
