CSS Selektorer

Nödvändiga att kunna i kursen

1. *

Universalselektorn

Denna selektor väljer alla element på sidan - all textfärg blir grön. 

* {
    color: green;
}

2. X

Elementselektorn

I detta fall blir alla länkar lila och alla paragrafer teckenstorlek 20px. 

a { color: purple; }
p { font-size: 20px; }

3. X, Y

Gruppselektorn

Separeras med ett kommatecken. Alla div's samt alla paragrafer får typsnittet Arial och storlek 22px. Fler än två element kan väljas. 

 div, p {
        font-family: Arial;
        font-size: 22px;
    }

4. #X

Selektor för id

Elementet med id="container" får färgen aqua som bakgrundsfärg. Id kan endast tillhör ett element. 

 #container {
            background-color: aqua;
        }

5 .X

Selektor för class

Alla element med class="gallery" får bredd 50% utifrån upptaget område av elementet. Class kan till skillnad från id sättas på flera element än ett. element. 

 .gallery {
                width: 50%;
            }

6. X Y

Arvselektor

Du använder denna när du vill komma åt ett elemen som är underelement till ett annat (parent/child). I detta fall elementet li i ul-listan ska ha backgrundsfärg violet.

Arvselektor - för att nå ett element inuti ett annat

 ul li {
            background-color: violet;
        }

Bra att kunna i kursen

7. X:visited

Pseudo-klass för besökta länkar

Pseudo-klasser betecknas med symbolen : . I detta fall blir alla av användaren besökta länkar gråa.

a:visited {
            background-color: grey;
        }

8. X:hover

Kallas för action pseudo class. När man gör mouse over element aktiveras regeln. I detta fall blir bakgrundfärgen lila när man "hovrar" över en länk. Bra att använd till knappar och menyer.

a:hover {
            background-color: yellow;
        }
