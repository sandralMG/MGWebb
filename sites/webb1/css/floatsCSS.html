Floats

Den klassiska metoden att skapa layout för webbsidor är att använda sig av floats. Idag finns det modernare alternativ somm CSS Flexbox och Grids, men floats är fortfarande vanligt förekommande och det är bra att kunna tillämpa den klassiska metoden för CSS layout.
I sitt urspringsläge visas alltid boxar i tur och ordning efter varandra, uppifrån till ner. I fallet då html-elementet är ett block-element, d.v.s de html-element som börjar på ny rad. En sådan layout blir inte särskilt spännande eller visuell. Med hjälp av floats kan element tas ut ur det "vanliga" layoutflödet och kunna placeras åt vänster (float: left;) eller höger (float: right).
