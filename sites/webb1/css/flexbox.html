CSS Flexbox

Flexbox är ett modernare sätt att skapa layout på webbsidor. Som man hör på namnet är flexbox ett flexibelt sätt att hantera boxar, både deras position och storlek, vilket ju är önskvärt för att kunna skapa layout.

Floats har funnits med sedan början av HTML/CSS, men för vissa situationer innebär det oänskade resultat som kräver omständiga workarounds. Som exempelvis "the clearfix-hack" eller att få kolumner bredvid varandra på samma höjd, etc (läs mer här om du är nyfiken). Det är fortfarande bra att ha kunskap om floats, men inom webbutveckling står vi inför ett skifte där de modernare formerna för layout som CSS Flexbox och Grids tar över allt mer. De når även tillräckligt stöd i olika webbläsare för kunna användas inom produktion. På sidan "Can I Use" kan du se hur Flexbox supportas i olika webbläsare. I dagsläget är det endast tidigare versioner av Internet Explorer som man bör implementera en fallback för.

Att lära sig Flexbox

Flexbox är inget eget språk utan det handlar om nya egenskaper i CSS som vi kan applicera på våra element. Det är enkelt att komma igång med Flexbox, man använder egenskapen   display:flex;  på det element man vill applicera reglerna för Flexbox på. Det kan exempelvis vara en meny, ett bildgalleri eller tre-kolumns layout på en sida. Det elementet som håller de olika elementen som ska "flexas" kallas för "flex container". En flex container är alltså det föräldraelement som innehåller olika flex items som skulle kunna vara navigationselement, bilder etc.

I exempelkoden nedan är div-elementet med class="parent" vår flex container och varje div-element med class="item" är flex items. Genom att sätta olika regler på vår .parent kan vi ordna boxarna inuti flex cobtainern så att de exempelvis blir centrerade i mitten, hamnar i en kolumn, får jämnt avstånd emellan o.s.v. 

 <div class="parent">
    <div class="item">1</div>
    <div class="item">2</div>
    <div class="item">3</div>
    <div class="item">4</div>
    <div class="item">5</div>
 </div>
    
.parent {
    border: 5px solid lightcoral;
    display: flex; /* Denna rad gör elementet till ett flex-element */
    justify-content: center; /* Denna rad centerar alla flex-items 
                                mot mitten. Det finns många olika 
                                flexboxkommandeon än denna att tillämpa. */
}
                                        
.item {
    margin: 5px;
    background: DeepSkyBlue;
    color: white;
    text-align: center;
    line-height: 50px;
    font-weight: 600;
}
