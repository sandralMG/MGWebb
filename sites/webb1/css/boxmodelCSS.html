Boxmodellen

Egenskaperna padding, border, margin sätts i en CSS-regel för det angivna element med en måttenhet, expempelvis antal pixlar i riktningarna top, right, bottom, left.
Det gäller för s.k block-element, d.v.s de element som börjar på en ny rad till skillnad från inline-element. Vi återkommer till vad som gäller för inline-elements.

Content

Innehållet som ligger inuti elementet, eller "boxen". Om man inte anger någon storlek, width eller height kommer default-värdet att användas vilket blir lika stort som föräldraelementets bredd och höjd.

Padding

Används för att skapa utfyllnad (tomrum) runtomkring innehållet och till en eventuell satt kantlinje (border). Padding är alltså bra för att skapa luftigare layout.

Border

Används för att skapa en kantlinje med en viss tjocklek, färg och utseende (ifylld, streckad, m. fl.).

Margin

Används för att skapa utrymme (tomrum) mellan olika box-element.
